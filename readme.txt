== Description ==

This plugins fires Google Analytics & Piwik events for the native Wordpress MediaElement video and audio player.
Adds the following events in Google Analytics & Piwik:

* play
* pause
* ended

An event look like this: wp-video, play,[filename]

* category: wp-video
* action: play
* label: The url of the corresponding media file

== Installation ==

* Install plugin
* Activate it
